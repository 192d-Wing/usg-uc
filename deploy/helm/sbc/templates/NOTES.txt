## Getting Started

1. Get the SIP service external IP:

   kubectl get svc {{ include "sbc.fullname" . }}-sip -n {{ .Release.Namespace }}

2. Get the API service endpoint:

   kubectl get svc {{ include "sbc.fullname" . }}-api -n {{ .Release.Namespace }}

3. Check the pod status:

   kubectl get pods -l "app.kubernetes.io/name={{ include "sbc.name" . }},app.kubernetes.io/instance={{ .Release.Name }}" -n {{ .Release.Namespace }}

4. View logs:

   kubectl logs -l "app.kubernetes.io/name={{ include "sbc.name" . }},app.kubernetes.io/instance={{ .Release.Name }}" -n {{ .Release.Namespace }}
